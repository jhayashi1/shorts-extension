# shorts-extension

Simple chrome extension to inject a button to redirect a youtube shorts video to the normal video player

## Setup

Install [`pnpm`](https://pnpm.io/installation)
and run: 
```bash
pnpm install
```

To run locally:
```bash
pnpm run start
```

To build:
```bash
pnpm run build
```
